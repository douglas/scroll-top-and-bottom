== Description ==
   Scroll Top and Bottom is created to scroll to top and bottom of the site. User can go to top and bottom of the site using this tool easily. This plugin is created with jQuery. Plugin is created by <cite><a href="http://shariarbd.com/" title="Md. Sahriar">Md. Shariar</a>.</cite>


== Installation ==

1. Upload `scroll-top-and-bottom` to the `/wp-content/plugins/scroll-top-and-bottom/` directory
(this can be done automatically via the WordPress 2.8 Plugin Browser/Installer interface or later)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Make sure that the wordpress function wp_head() is included on your theme.
